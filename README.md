# AI Workspace

Shared collaboration space for Atlas and Tyler.

## Structure

```
ai-workspace/
├── HANDOFF.md    # Current task state (read before every handoff)
├── TASKS.md      # Shared task board
├── context/      # Project notes, meeting summaries
└── skills/       # Skills we build together
```

## Protocol

1. **Before starting work:** `git pull` and read HANDOFF.md
2. **After completing work:** Update HANDOFF.md and TASKS.md, then `git push`
3. **Handoffs:** Tag each other in Discord, reference this repo
